# 🚖 Ola Cab CLI Project (Java)

## 📌 Description
A command-line based cab booking system built using Java.

## 🚀 Features
- Select cab type (Mini, Sedan, SUV, Luxury)
- Displays price
- Uses OOP concepts

## 🛠 Technologies
- Java
- OOP (Inheritance, Polymorphism)

## ▶️ Run Project
javac OlaCab.java  
java OlaCab

## 📚 Concepts Used
- Inheritance  
- Method Overriding  
- Polymorphism
